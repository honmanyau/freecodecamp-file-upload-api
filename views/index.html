File Metadata API

This API takes a list of files, upload it to the server, and returns a JSON string with the metadata about the files uploaded. The metadata returned for each file includes its name, size, encoding, its MIME type. The number of files that can be uploaded at a time is set to 10, with a file size limit of 2 MB each. These limits are arbitrarily set on the client side only.

It is worth noting that, in this project, the size of a file is displayed before it's uploaded because this information is readily available from the <input type="file" > element as soon as files are selected, which arguably renders this particular API rather pointless and wasteful as everything can be handled on the client side.

Example Output:

{"files":[{"filename":"Alicepng","bytesize":1377,"encoding":"7bit","mimetype":"image/png"}]}
